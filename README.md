#### magicball.rb 
*Руби файл с программой Волшебный шар.*

[Волшебный шар на Вики](https://ru.wikipedia.org/wiki/Magic_8_ball)

Чтобы запустить программу выполните следующие действия:
 1. Откройте терминал:
 (в Убунту ```"ctrl + alt + T"```)
 2. Запустить файл можно введя программу:
 ```
 ruby magicball.rb
 ```
